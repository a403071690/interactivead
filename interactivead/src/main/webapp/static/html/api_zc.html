接口文档说明

参数名 类型 备注
请求方法 http get/post都可以
请求参数格式 json格式(application/json) 查询和删除参数放入url,保存和更新用post将json数据放入请求体
返回格式 json格式

查询公共参数

参数名 类型 备注
pageNum Integer 分页查询的第几页(分页查询必传该参数)
pageSize Integer 每页显示的条数
orderProperty String 根据该属性进行排序
orderDirection String desc/asc(降序/升序)
fullTextSearchValue String 全文搜索(正则表达式匹配)
参数名+"Start" 查询大于或等于某Date 如查询创建时间大于或等于某时间则传createTimeStart="yyyy-MM-dd HH:mm:ss"
参数名+"End" 查询小于某Date 如查询创建时间小于某时间则传createTimeEnd="2018-1-30 23:59:59.9"(yyyy-MM-dd HH:mm:ss.SSS)S为毫秒
参数名+"List" 值为多个或匹配 如查询类型为a或者b的数据则传type=['a','b']
值为一对多关联表的参数规则: 参数key为表名,值为json数组，数组内对象的key为字段名

参数名 类型 备注

保存或修改接口公共参数

参数名 类型 备注

返回公共字段

参数名 类型 备注

查询接口 host+"/duiCreditsConsume/select"
删除接口 host+"/duiCreditsConsume/delete"
保存或修改接口 host+"/duiCreditsConsume/saveOrUpdate"
参数名 类型 备注
id String 数据唯一id
uid String 用户唯一性标识，唯一且不可变
credits Long 本次兑换扣除的积分
itemCode String 自有商品商品编码(非必须字段)
appKey String 接口appKey，应用的唯一标识
timestamp Date 1970-01-01开始的时间戳，毫秒为单位。
description String 本次积分消耗的描述(带中文，请用utf-8进行url解码)
orderNum String 兑吧订单号(请记录到数据库中)
type String 兑换类型：alipay(支付宝), qb(Q币), coupon(优惠券), object(实物), phonebill(话费), phoneflow(流量), virtual(虚拟商品), littleGame（小游戏）,singleLottery(单品抽奖)，hdtoolLottery(活动抽奖),htool(新活动抽奖),manualLottery(手动开奖),ngameLottery(新游戏),questionLottery(答题),quizzLottery(测试题),guessLottery(竞猜) 所有类型不区分大小写
facePrice Integer 兑换商品的市场价值，单位是分，请自行转换单位
actualPrice Integer 此次兑换实际扣除开发者账户费用，单位为分
ip String 用户ip，不保证获取到
waitAudit String 是否需要审核(如需在自身系统进行审核处理，请记录下此信息)
params String 详情参数，不同的类型，返回不同的内容，中间用英文冒号分隔。(支付宝类型带中文，请用utf-8进行解码) 实物商品：返回收货信息(姓名:手机号:省份:城市:区域:详细地址)、支付宝：返回账号信息(支付宝账号:实名)、话费：返回手机号、QB：返回QQ号
sign String MD5签名
backStatus String 扣积分结果状态，回复ok或者fail （不要使用0和1）
backErrorMessage String 出错原因
backBizId String 开发者的订单号(兑吧会判断该订单号的唯一性，注意区分测试和正式对接时的订单号，以免重复)
backCredits Integer 用户积分余额
